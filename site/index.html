// questions

Straight answers.

Does M1K3 send my data to a server?

No. M1K3 runs entirely on your Mac — inference, voice, embeddings, and storage are all on-device. Zero bytes of your conversations, documents, or calls go to any server. The optional web-search tool is off by default and clearly toggled.

What AI models does it run?

Google Gemma locally via MLX-Swift on Apple Silicon, with Apple Foundation Models handling fast turns. Hot-swap runtimes in Settings, mid-conversation.

Which Macs are supported?

Apple Silicon Macs on macOS 26 Tahoe. M1K3 is a native SwiftUI app with Metal-accelerated inference — not an Electron wrapper.

How is it different from Ollama or ChatGPT?

Ollama is a local model server; ChatGPT is a cloud service. M1K3 is a complete on-device companion: local models plus live voice, a knowledge graph over your documents, encrypted call memory, an agent with tools, and an MCP server — one sandboxed native app.

Can Claude use my M1K3 knowledge?

Yes. M1K3 ships an MCP server exposing search_knowledge, list_documents, and get_document — Claude and other MCP clients can query your local knowledge base, locally.

Is it free?

Free and open source, made by Round Tower in Ireland. Build from source today, or join the launch list for the packaged app.
