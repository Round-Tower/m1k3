## 🛠️ Tools for Ripping Your Own Games

**IMPORTANT:** Only rip games you legally own!

### Legally Rip Your Own Games:
1. **Ninja Ripper** - http://www.ninjaripper.com/
   - Captures 3D models from ANY game
   - Export to .obj → Convert to .glb

2. **3D Ripper DX** - http://www.deep-shadows.com/hax/3DRipperDX.htm
   - DirectX game ripper

3. **Blender Import Plugins**
   - N64 models: [github.com/Thar0/N64-Tools](https://github.com/Thar0/N64-Tools)
   - PS1 models: Various community tools

### Conversion Workflow:
```
Your Game → Ripper Tool → .obj/.fbx
  ↓
Blender (import)
  ↓
File → Export → glTF 2.0 (.glb)
  ↓
Drag into M1K3! 🎉
```

**Always Legal:**
- CC0 models (public domain)

- GPL/MIT licensed game assets
- Models you create yourself
- Models from games you own (personal use)

**Never Share Commercially:**
- Copyrighted game assets
