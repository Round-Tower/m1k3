# 🎨 Open Source 3D Model Library

Curated list of awesome free/open-source .glb models you can drag-and-drop into M1K3!

## 🚀 How to Use

1. **Download** - Right-click → Save As on any model link below
2. **Drag & Drop** - Drag the .glb file onto the M1K3 avatar viewer
3. **Or Click** - Use the "📁 UPLOAD .GLB" button in model selector

## 💡 Pro Tips

### For Best Results:
- **Animated models** - M1K3 auto-detects animations!
- **File size** - Under 10MB for smooth performance
- **Rigged models** - Skeletal animation support
- **PBR materials** - Looks best with M1K3's lighting

### Troubleshooting:
- **Model too big/small?** - M1K3 auto-normalizes scale
- **No animations?** - Static models work too, just no motion
- **Won't load?** - Check it's .glb or .gltf format (not .fbx, .obj, etc.)
