Used in M1K3 as the optional "Dyslexia-friendly" reading mode for assistant
replies — chosen because the typeface's weighted bottoms and distinct letterforms
help some dyslexic readers track the line. Reading mode is opt-in (Settings ›
Reading); the default remains the system font.
