Thank you for reviewing M1K3.

NO ACCOUNT REQUIRED
M1K3 has no sign-in, no account, and no demo credentials. Just launch and use it.

FIRST LAUNCH — ONE-TIME MODEL DOWNLOAD
On first run, M1K3 downloads an on-device AI model (a few hundred MB) so it can run
entirely locally afterward. This download is the ONLY network request the app needs
to function. Please allow it to finish on first launch before testing chat/voice.
After that, the assistant runs fully offline on the user's Mac.

NETWORK USAGE — IN FULL
M1K3 is a privacy-first, on-device assistant. Outbound network use is limited to:
  1. The one-time model download described above.
  2. An OPTIONAL web search, which is OFF by default and must be explicitly enabled
     by the user per request (a visible toggle). When off, the app makes no search
     or content requests.
There is no analytics, no tracking, no telemetry, and no user data sent to any server.
Chats, notes, PDFs, and voice audio stay on the device.

PERMISSIONS
  • Microphone — used only for on-device speech-to-text in voice mode. Audio is
    transcribed locally and is not transmitted.
  • Files/Documents — used only when the user drags in a note or PDF for the
    assistant to reference; processed locally.

HOW TO EXERCISE THE APP
  1. Complete first-run onboarding and let the model download finish.
  2. Send a chat message (text), then try voice mode.
  3. Drag in a short note or PDF, then ask a question about it.
  4. Open the Memories view and confirm "forget" removes an item.
  5. Try voice output (system voice and the neural voice).

Everything runs on-device; the only network use is the first-run model download and
the optional, explicitly enabled web search. Happy to clarify anything — thank you.
